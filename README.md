# TPM Vibe-Coding Tool 🚀

A simple, fast-deployed decision-making tool built to explore the "vibe-coding" workflow. 

## 📝 About the Project
This project serves as my playground for learning Git, GitHub workflows, and AI-assisted development. As a TPM, I'm building this to bridge the gap between high-level system design and hands-on implementation. 
![App Screenshot](Screenshot.png.png)

## 🛠 Built With
* **HTML/CSS/JS:** Core building blocks.
* **GitHub:** Version control and remote storage.
* **GitHub Pages:** Automated deployment.
* **"Vibe Coding":** AI-assisted development workflow.

## ⚙️ How it works (The Vibe Loop)
1. **Develop:** Use VS Code to write/refactor code.
2. **Commit:** Use GitHub Desktop to track changes and save snapshots.
3. **Deploy:** Push to `main` and watch GitHub Pages publish the update automatically.
